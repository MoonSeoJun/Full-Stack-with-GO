GO언어의 자료형
===============

+ ## var
```
var s string = "MunSeoJun"
```
기본적으로 go언어에서는 var 키워드를 사용해 변수를 선언한다.

-----------------------------

+ ## :=
```
s := "MunSeoJun"
```
함수 안에서 변수를 선언하고 초기화할 때는 := 를 사용한다.
이를 타입 추론이라고 한다.

----------------------------------

+ ## 자료형의 종류

### 1. bool : true 혹은 false를 가짐
### 2. string : Byte형의 조합으로 문자를 저장한다.
### 3. byte : 바이트를 나타낸다. uint8과 같다.
### 4. int : 부호를 갖는 정수형이다.
### 5. float : 실수형을 나타냄.
### 6. complex : 복소수로 복잡한 수학 연산시 사용된다.
